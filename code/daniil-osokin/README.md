**Выполнил:**

## Задание

Реализовать класс, осуществляющий поиск в тексте по регулярным выражениям:

 * {n} - "ровно n раз"
 * ? - "ноль или одно"
 * ^ - "начало строки"

Под текстом подразумевается множество строк конечной длины.

Подробное описание можно найти в статье
